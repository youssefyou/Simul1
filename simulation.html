Notre but ici est de raliser un graphique qui montre lvolution de quatre variables alatoires qui suivent une loi normale centre rduite.

Simulation des variables alatoires

Commenons dabords par la cration de notre jeu de donnes.

set.seed(1)
newdata <- data.frame(Var1= rnorm(40), Var2= rnorm(40), Var3= rnorm(40),
                      Var4= rnorm(40), index=1:40)

Reprsentation avec plot R-base:

La reprsentation graphique avec la fonction plot de R-base est comme suit:

plot(newdata$Var1, col=1, ylim= c(-2.5, 3), ylab= "value", type="l", lwd=2)
lines(newdata$Var2, col=2, lwd=2)
lines(newdata$Var3, col=3, lwd=2)
lines(newdata$Var4, col=4, lwd=2)
legend("topright", legend = c("var1", "var2", "var3", "var4"),
       col= 1:4, lwd=2,ncol=4, cex=0.8, bty="n", bg="white")

Si on veut obtenir un graphe dune qualit suprieure on peut utiliser le package ggplot2.

Reprsentations avec ggplot2:

Pour raliser notre graphe avec le package ggplot2 nous allons dabords transform notre jeu de donnes.

library(reshape2)
Mnewdata <- melt(newdata, id = "index")

Enfin, ci- dessous quelques graphiques raliss avec la fonction ggplot:

library(ggplot2)
ggplot(Mnewdata, aes(index, value, colour=variable)) +
    geom_line(size=1) +
    scale_color_brewer(palette="Spectral")

ggplot(Mnewdata, aes(index, value, colour=variable)) +
    geom_line(size=1) +
    scale_color_brewer(palette="Spectral") + theme_bw()

ggplot(Mnewdata, aes(index, value, colour=variable)) +
    geom_line(size=1) +
    scale_color_brewer(palette="Set1")

ggplot(Mnewdata, aes(index, value, colour=variable)) +
    geom_line(size=1) +
    scale_color_brewer(palette="Dark2")

ggplot(Mnewdata, aes(index, value, colour=variable)) +
    geom_line(size=1) +

    theme_classic()
